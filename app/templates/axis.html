Axes

Axes are, by default, added to every histogram and scatter plot tracks. There is no need for them when displaying a heatmap or tiles.

Any modifications regarding the format of axes should be made in the <axes> block from the axis.conf file.


No axis

You might want to remove the axes on some of your tracks. In this case, the alteration must be made in specific.conf rather than in axis.conf. In the appropriate <plot> block, remove or comment the line <<include axis.conf>>.

If you want to remove all axes from all tracks, change the display paramater of the <axes> block:

<axes>
  show = no
  <axis>
    color = vlgrey
    spacing = 0.1r
  </axis>
</axes>

Spacing

To define the ticks on y-axis, you must define the spacing between them. The interface set it by default to spacing = 0.1r. The r means that the spacing will be relative to the track dimension. Using a smaller value will augment the number of ticks, while a bigger value will dimish the numebr of ticks.

You can also define the spacing in units:

spacing = 5

spacing = 10

Tip: it is easier to define unitary spacing when you know or have set minimum and maximum values. (in the example, the values range from 0 to 50).


Color & Size

When you generate a Circos, the axis will be by default grey and of size 1. To alter them, you can change the color name on the line color = grey and/or the integer on the line thickness = 1.

color = vdred
thickness = 1

color = vdred
thickness = 5
