Histogram

The first step in altering a track represented by a histogram is to locate its <plot> block in specific.conf.

You should find the line type = histogram in that block.


Colors

By default, the bins will be filled white. You can easily change it by replacing white with another color name:

fill_color = white

fill_color = green

It is much easier to differentiate bins this way, especially when you have positive and negative values. Another way to do it would be to identify values greater than 0 in blue, and values smaller than 0 in red.

<plot>
  type = histogram
  svgclass = histogram
  file = histo.txt
  r0 = 0.500r
  r1 = 0.950r
  orientation = out
  fill_color = white
  <<include axis.conf>>
  <rules>
    <<include svg_rule_track.conf>>
    <rule>
      condition = var(value) > 0
      fill_color = blue
    </rule>
    <rule>
      condition = var(value) < 0
      fill_color = red
    </rule>
  </rules>
</plot>

You can choose any color to differentiate your positive values from the negative ones.


Min & Max

The interface does not define a specific minimum and/or maximum values for your histogram. Instead, it will take the smallest and largest value from your data set, and use them as the min and max. It can sometime be hard to analyze data when there is a lot of small values and only a few large ones: the y-axis will be as large as the largest value. You can therefore set your own min and max within the <plot> block:

min = 0

max = 0

min = -40

max = 40


Background

Changing the background color can sometime be useful. If you want to do so, you will first have to define a new general block, <backgrounds>. This block is within <plot> and will contain smaller blocks, <background>.

<plot>
  type = histogram
  svgclass = histogram
  file = histo.txt
  r0 = 0.500r
  r1 = 0.950r
  orientation = out
  fill_color = white
  <<include axis.conf>>
  <rules>
    <<include svg_rule_track.conf>>
  </rules>
   <backgrounds>
   </backgrounds>
</plot>
<backgrounds>
  <background>
    y0 = 
    y1 = 
    color = 
  </background>
</backgrounds>

You can define as many <background> blocks as you want.

y0 and y1 represent the y-axis range to be colored.

color takes as value a color name. 




<backgrounds>
  <background>
    y0 = 0
    y1 = 500
    color = vlblue
  </background>
  <background>
    y0 = 0
    y1 = -300
    color = vlred
  </background>
</backgrounds>
